# Frever Fitness v7 Stable Recovery-first multi-page build for GitHub Pages and Firebase. ## Upload Upload the contents of this folder directly to the root of the Fitness2 repository. Keep the existing CNAME file. Required root files include: - index.html - home.html - workout.html - routines.html - history.html - pbs.html - body.html - food.html - classes.html - timer.html - exercises.html - settings.html - app.js - styles.css - assets/frever-logo.svg ## Important checks - `app.js` should be approximately 85 KB and begin with Firebase imports. - `assets/frever-logo.svg` is the small panda SVG. - Do not rename or flatten the `assets` folder. - No Firebase Console changes are required.

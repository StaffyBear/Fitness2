# Frever Fitness v6.1

Stable clean-URL rebuild based on the last working Firebase version.

## Upload to GitHub Pages

Upload the **contents of this folder** directly to the repository root. Keep the existing `CNAME` file.

The repository root must contain:

- `index.html`
- `app.js`
- `styles.css`
- `assets/`
- `home/`
- `workout/`
- `routines/`
- `history/`
- `pbs/`
- `body/`
- `food/`
- `timer/`
- `exercises/`
- `settings/`

Do not upload the enclosing `frever-fitness-v6.1` folder itself.

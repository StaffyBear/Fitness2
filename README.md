# Frever Fitness v8.4 — Workout Flow

Upload the contents of this folder to the root of the Fitness2 test repository. Keep the existing CNAME file.

Changes:
- Search box and exercise information button in workout exercise cards.
- Unlimited saved sets without the “0 of 3” counter or “+ Set” button.
- Workout session timer starts with the first saved set or when a routine starts, persists through refresh, and saves duration to History.
- Workout History sorted by workout date, with weekly and monthly summaries.
- Routines can be sorted by day, name, recently updated or last performed; day is a fixed selector.
- Food Today view moves one day at a time with the arrows; Planner/Week still move by week.
- Local Classes filter by day of week.
- Existing round-history and login fixes retained.
